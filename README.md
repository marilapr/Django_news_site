Создан простой новостной сайт с регистрацией пользователей, управлением новостями!

На данном сайте можно: 
- Посмотреть новости на главной странице
- Регистрация и авторизация пользователей
- Добавление новостей (только для авторизованных пользователей)
- Редактирование и удаление своих новостей
- Пагинация (6 новостей на страницу)
- Темная/светлая тема оформления

 Backend: Django
База данных: SQLite3
