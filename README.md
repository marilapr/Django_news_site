Создан простой новостной сайт с регистрацией пользователей, управлением новостями!

На данном сайте представлено: 
- Просмотр новостей на главной странице
- Регистрация и авторизация пользователей
- Добавление, редактирование и удаление новостей (только для авторизованных пользователей)
- Отображение по блокам из 6 новостей
- Темная/светлая тема оформления

Backend: Django
База данных: SQLite3
